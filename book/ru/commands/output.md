# Вывод данных на экран

## Команда head

Команда **head** по умолчанию выводит первые 10 строк файла.

<ins>**Ключи**</ins>:

* **-с**– позволяет указать точное количество байт, которое необходимо вывести на экран;
* **-n** – позволяет указать точное количество строк, которое необходимо вывести на экран.

**Создание файла с которым будем работать:**

*используем команду echo*

```
$ echo “текст” > наш_файл
```

**Вывести на экран первые 10 строк файла.**

```
$ head /FileHead
```

![head](../img/commands/output/1.png) 

**Вывести на экран первые 100 байт файла.**

```
$ head –c 100 /FileHead
```

![head](../img/commands/output/2.png) 

**Вывести на экран первые 20 строк файла.**

```
$ head –n 20 /FileHead
```

![head](../img/commands/output/3.png)

## Команда tail

Команда, обратная по действию команде **head** — **tail**. Используя команду **tail**, вы можете увидеть последние десять строк файла. Это может оказаться полезным, если, например, вы хотите просмотреть последние строки файла журнала системных сообщений. Также при помощи программы tail можно наблюдать за изменением журнала.

<ins>**Ключи:**</ins>

* **-n N (--lines=N)** – позволяет указать точное количество строк, которое необходимо вывести на экран (N – число);
* **-f**  - показывает последние 10 строк файла. Команда продолжает показывать новые строки по мере их добавления к выбранному файлу. Для окончания просмотра нажмите `Ctrl + C`.  Для просмотра системных файлов необходимо быть суперпользователем. 

**Создание файла с которым будем работать:**

*используем команду echo*

```
$ echo “текст” > наш_файл
```

**Вывести последние 10 строк файла.**

```
$ tail FileTail
```

![Tail](../img/commands/output/4.png)

**Вывести на экран последние 5 строк файла.**

```
$ tail --lines=5 FileTail
```

![Tail](../img/commands/output/5.png)

**Обновлять информацию по мере появления новых строк в файле.**

```
$ tail –f  FileTail
```

![Tail](../img/commands/output/6.png)

Отличительной чертой команды является ее возможность не завершаться по достижении конца вывода, а ожидать появления новых данных.

## Команда less

Команда less осуществляет поиск по данным в файле.

<ins>**Ключи:**</ins>

* **-р текст** - указывает первую строку, в которой найден текст;
* **-N** - показывать номера строк;
* **-s** - сжимает несколько пустых строк в одну пустую строку.

**Работа с файлом осуществляется с помощью команд:**

| Команда | Назначение |
|---|---|
| k и j или стрелки вверх и вниз | Прокрутка текста на строку назад. Прокрутка текста на строку вперед.|
| стрелки вправо и влево | Прокрутка текста влево или вправо на позицию табуляции. |
| PgUp или SPC | Прокрутка текста вперед на страницу. |
| PgDn | Прокрутка текста назад на страницу. |
| Ctrl-l | Перерисовка экрана. |
| /pattern | Поиск строк, содержащих образец, вперед по тексту внутри файла |
| < | Перемещение к началу файла. |
| > | Перемещение к концу файла. |
| G | Перейти к концу документа. |
| q | Прекращение просмотра и выход из less. |


**Создание файла с которым будем работать:**

```
$ echo “текст” > наш_файл
```


**Вывести на экран текстовый файл.**

```
$ less FileLess
```

![less](../img/commands/output/7.png)

**Найти в файле текст «Там».**

```
$ less –p Там FileLess
```

![less](../img/commands/output/8.png)


**Вывести текст с пронумерованными строками.**

```
$ less –N FileLess
```

![less](../img/commands/output/9.png)

**Сжать несколько пустых строк в одну.**

Добавим пустые строки в наш файл:

```
$ echo “текст” > наш_файл
```

```
$ less –s FileLess
```

![less](../img/commands/output/10_1.png) ![less](../img/commands/output/10_2.png)

**Найти подстроку «без» внутри нашего файла.**

Откроем наш файл командой

```
$ less FileLess
```

Введем в открывшемся файле команду

/без

![less](../img/commands/output/11_1.png) ![less](../img/commands/output/11_2.png)


## Команда more

Команды **more** и **less** тесно связаны и функционально подобны. Основное назначение этих команд - вывод на экран или в окно текста большого объема по частям, чтобы его можно было листать в обоих направлениях, а также поиск в этом тексте.

**Команда more** – команда постраничной или построчной прокрутки большого текста на экране и поиска в этом тексте.

<ins>**Ключи:**</ins>

* **-uN** – вывод первых N строк;
* **-s** - сжать каждый пустой блок пустых строк в одну строку.

Работа с файлом осуществляется с помощью команд:

| Команда | Описание |
|---|---|
| SPACE | пролистать один экран |
| d | пролистать пол экрана |
| /pattern | пролистать до строки содержащей подстроку pattern |
| n | повторить поиск подстроки |
| = | показать номер текущей строки |
| :f | напечатать имя текущего файла и номер строки в нем |


**Создание файла с которым будем работать:**

используем команду echo

```
$ echo “текст” > наш_файл
```

**Вывести на экран текстовый файл.**

```
$ more FileMore
```

![more](../img/commands/output/12.png)

**Вывести первые 5 строк файла.**

```
$ more –u5 FileMore
```

![more](../img/commands/output/13.png)

**Сжать несколько пустых строк в одну.**

```
$ more –s FileMore
```

![more](../img/commands/output/14.png)

## Команда cat

Команда **cat** – вывести файл на stdout, также существует возможность перенаправить потоки ввода - вывода с помощью >, <.

<ins>**Ключи:**</ins>

* **-b** – нумеровать только непустые строки;
* **-E** - показывать символ $ в конце каждой строки;
* **-s** - удалять пустые повторяющиеся строки.

**Создать файл с помощью перенаправления потока вывода.**

```
$ cat > FileCat
```

Ожидая ввод от пользователя, наберите необходимый текст и нажмите Ctrl + D, чтобы выйти. Текст будет записан в указанный файл. 

![less](../img/commands/output/15_1.png) ![less](../img/commands/output/15_2.png)

Посмотрим содержимое файла, набрав команду 

```
$ cat FileCat
```

![less](../img/commands/output/15_1.png) ![less](../img/commands/output/15_2.png)

**Добавить в наш файл содержимое другого файла**

Для начала создадим новый файл тем же способом что и первый. Затем, с помощью команды 

```
$ cat File1.txt >> File2.txt
```

добавим содержимое второго файла к первому.

![less](../img/commands/output/16.png)

Также можно перезаписать файл командой

```
$ cat File1.txt > File2.txt
```

![less](../img/commands/output/17.png)

**Вывести на экран файл.**

```
$ cat FileCat
```

![less](../img/commands/output/18.png)

**Создать файл с записью в него содержимого нескольких других файлов.**

```
$ cat File1.txt File2.txt > File3.txt
```

![less](../img/commands/output/19.png)

**Написать многострочный набор данных с отступами и специальными символами.**

```
$ cat > File2 << EOF
EOF – метка конца файла.
```

![less](../img/commands/output/20.png)

**Пронумеровать только непустые строки в файле.**

```
$ cat --number-nonblank FileCat
```

![less](../img/commands/output/21.png)

**Вывести файл со знаком $ в конце каждой строки.**

```
$ cat --show-ends FileCat
```
![less](../img/commands/output/22.png)

**Удалить пустые повторяющиеся строки.**

``
$ cat --squeeze-blank FileCat
``

![less](../img/commands/output/23.png)

## Команда tee

Команда **tee** перенаправляет входной поток в файл и в стандартный вывод, т.е. то, что мы введем запишется в файл, который мы укажем, и выведется на экран.

<ins>**Ключи:**</ins>

* **-а** - дописать выходящие данные в конец файла, не переписывая его содержимого.

**Создать файл.**

```
$ tee FileTee
```

Посмотрим получившийся файл командой cat:

![less](../img/commands/output/24.png)


**Дописать данные в конец файла, не переписывая его.**

```
$ tee –a FileTee
```

![less](../img/commands/output/25.png)
