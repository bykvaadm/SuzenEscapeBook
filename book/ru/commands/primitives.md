# Примитивные команды с которых стоит начинать изучение

### Change Directory (cd)

Нужен для бла-бла-бла...

Представим себе следующее расположение директорий:

```
.
└── tmp
    └── cd
        ├── dir1
        │   ├── dir4
        │   ├── dir5
        │   └── dir6
        ├── dir2
        │   ├── dir7
        │   ├── dir8
        │   └── dir9
        └── dir3
            ├── dir10
            ├── dir11
            └── dir12
```

где . - **корневая директория**, котора в Linux обозначается символом /
